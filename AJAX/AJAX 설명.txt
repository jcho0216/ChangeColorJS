API: 이용자가 단순히 애플리케이션의 결과나 사용자 인터페이스만 수동적으로 제공받는 것이 아니라 직접 응용
프로그램을 개발할 수 있도록 공개된 개발용 프로그래밍 라이브러리.

지도 데이터를 불러오기 위해서는 서버와 통신이 필요하다. 사전 학습을 통해 서버에 있는 데이터를 요청하는방법

AJAX는 "Asynchronous Javascript and XML"의 줄임말, 자바스크립트를 이용해 서버와 데이터를 주고받는 방식

AJAX로 하는일
- TEXT, HTML, XML, JSON 형식으로 작성된 문서 데이터를 HTTP 프로토롤을 통해 서버에 요청할 수 있습니다.
- 구글 맵이나 유튜브 서비스와 같이 웹 페이지의 특정 영역에만 필요한 내용을 갱신할 수 있습니다.
